# 树兰（杭州）医院就诊指南

## 门诊就诊提示

- 门诊常规时间：上午 8:00-12:00，下午 13:30-17:00
- 夜门诊：以医院当期排班为准，耳鼻喉科夜门诊常见时段为 17:00-22:00
- 医保患者就诊时，建议携带身份证、医保卡和相关证历资料
- 自费患者就诊时，建议携带身份证和就诊卡或杭州市健康卡

## 到院后可以在哪里办理业务

- 门诊一楼大厅：一站式服务中心、预检分诊、预约咨询、医保咨询
- 自助服务区：挂号、取号、缴费、报告打印、价格查询
- 用户服务中心：可处理预约、分诊、部分便民服务和咨询

## 急诊说明

- 急诊全年 365 天、24 小时开放
- 急诊主要包括预检区、内外科诊疗区、输液室、抢救室、急诊换药间和急诊化验室
- 急诊可处理除 14 周岁以下儿科和烧伤外科之外的大部分急诊急救需求，夜间口腔科、皮肤科除外

## 急诊常见流程

预检分诊 -> 挂号与建卡 -> 医生接诊 -> 必要检查 -> 继续诊疗 -> 门诊随访、收入住院、急诊留观或转院

## 适合直接急诊的情况

- 突发胸痛、呼吸困难、意识障碍
- 突发肢体无力、言语不清、口角歪斜
- 严重外伤、大出血、中毒、高热惊厥等
