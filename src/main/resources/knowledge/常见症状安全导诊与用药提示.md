# 常见症状安全导诊与用药提示

## 文档用途

本资料用于辅助智能问答处理“头痛、鼻塞、发热、咳嗽、咽痛、想去药店买药”等高频咨询。它只作为导诊和安全提醒资料，不替代医生诊断、药师审方或药品说明书。

## 回答原则

- 先判断是否存在需要及时就医的危险信号，再讨论居家观察或药店咨询。
- 不直接承诺某一种药一定适合用户；需要结合年龄、孕产状态、基础病、过敏史、正在使用的药物和症状持续时间。
- 不建议自行使用抗菌药物。普通感冒、流感、新冠和其他呼吸道感染症状可能相似，必要时应通过病原学检测或医生评估区分。
- 对儿童、老年人、孕产妇、免疫功能低下者、慢性心肺肾肝病或糖尿病患者，应降低就医阈值。
- 提醒用户避免重复服用含相同成分的复方感冒药，尤其要注意退热镇痛成分叠加。

## 需要尽快就医或急诊评估的情况

- 高热不退、精神反应差、意识改变、抽搐、明显脱水。
- 呼吸急促、胸闷胸痛、口唇发紫、静息状态下气喘或血氧下降。
- 剧烈头痛、突发单侧肢体无力、言语不清、视物异常、颈项强直。
- 持续发热超过 3 天，或症状快速加重。
- 儿童出现拒食、喂养困难、呼吸费力、持续高热或惊厥。
- 老年人、孕产妇和有慢性基础病的人群出现流感样症状并明显不适。

## 头痛、鼻塞、发热、咳嗽类咨询的建议结构

1. 先追问：年龄、体温、症状持续多久、是否咳嗽咽痛、是否肌肉酸痛、是否接触流感或新冠患者、是否有基础病、是否怀孕、是否药物过敏。
2. 如果症状轻、无危险信号：建议休息、补液、保持通风，药物选择应咨询药师并按说明书使用。
3. 如果处于流感流行季，且有发热、头痛、肌肉酸痛、咽痛、咳嗽、鼻塞流涕等表现：提示可能是流感或其他呼吸道感染，建议关注高危因素，必要时及时就医检测。
4. 如果用户问“去药店配什么药”：可以建议向药师说明完整症状和既往病史，不建议自行叠加多种复方感冒药；儿童用药必须按年龄和体重选择，避免使用成人药替代。
5. 如果用户已有医生处方：提醒按医嘱执行，不自行加量、停药或换药。

## 慢病与生活方式咨询的边界

- 高血压、糖尿病、高脂血症、肥胖相关问题，可以结合国家卫健委食养指南和诊疗指南给出饮食、运动、体重管理和复诊提醒。
- 高尿酸血症与痛风相关问题，可以提示饮食管理、规律复诊和避免自行调整降尿酸药物；急性关节红肿热痛、反复发作或合并肾病时应建议就医评估。
- 慢性肾脏病相关问题，应强调个体化营养方案和复诊监测，不根据聊天内容直接给出蛋白、钠、钾、磷摄入处方。
- 慢阻肺、长期咳嗽咳痰、喘息相关问题，应关注吸烟史、活动后气促、急性加重和血氧情况，明显呼吸困难或低氧时应及时就医。
- 不根据一次聊天直接调整降压药、降糖药、调脂药或减重药剂量。
- 出现血压显著升高并伴胸痛、气促、神经系统症状，或血糖异常并伴意识改变、呕吐、脱水等情况，应提示及时就医。
