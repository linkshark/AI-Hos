# 树兰（杭州）医院重点科室与学科能力

> 资料整理时间：2026-04-03  
> 说明：以下以医院官网科室导航和科室介绍页为主，优先保留有明确公开描述的重点能力。

## 1. 肝胆胰外科

- 科室于 2015 年 12 月成立，由郑树森院士领衔。
- 官网提到医疗团队共有医师 42 人，约 40% 拥有博士学位，开放 4 个病区、274 张床位。
- 特色能力集中在：
  - 肝胆胰良恶性疾病外科治疗
  - 肝移植及多器官联合移植
  - ERCP/EUS 诊治中心
  - 肝移植术后胆道并发症处理
- 官网专家页可见代表性专家包括李启勇、张武、杨喆、罗卫庆、李建辉、丁松明等。

## 2. 肝病暨感染科门诊 / 感染病科

- 是树兰医院最强辨识度科室之一，与李兰娟院士团队直接关联。
- 科室公开专家包括李兰娟、汤灵玲、高海女、朱梦飞、汪鑫、盛国平、陈平、邹鹏飞、戴霞红等。
- 从专家页和门诊排班可见，该方向覆盖：
  - 发热性疾病
  - 各类肝病、肝衰竭
  - 人工肝治疗
  - 微生态诊疗 / 菌群相关研究
  - 肝移植围手术期感染与管理

## 3. 肾脏病科

- 寿张飞院长兼任肾脏病科主任。
- 官网明确的核心能力包括：
  - 肾移植及移植并发症诊治
  - 移植术后长期随访
  - 血液透析、血滤、CRRT、血液灌流、血浆置换、免疫吸附
  - 腹膜透析置管与常规腹透
  - 自动化腹膜透析（APD）和远程居家智慧腹透管理
- 官网还提到高致敏患者肾移植、低龄婴儿供肾整块肾移植等疑难场景。

## 4. 呼吸与危重症医学科

- 公开资料显示科室围绕肺部感染病原筛查、抗生素耐药机制、肺癌早诊和精准治疗开展研究。
- 强调对以下问题的诊治经验：
  - 社区获得性肺炎
  - 支气管哮喘
  - 慢阻肺
  - 重症肺炎
  - 呼吸衰竭
  - 弥漫性肺疾病
- 官网列出的科室专家包括李宗煜、祝艳。

## 5. 重症医学科（综合 ICU）

- ICU 在开院后即设立，重症医学科于 2019 年 12 月成立。
- 官网信息显示核定床位 35 张，实际开放床位 55 张。
- 能力覆盖：
  - 各类休克
  - 多器官功能衰竭
  - 大器官移植术后监护
  - 严重创伤
  - 心跳呼吸骤停后的高级生命支持
- 该科是医院重大、疑难、高危和大器官移植手术的重要保障单元。

## 6. 急诊科

- 官网称急诊科是医院建立最早的科室之一。
- 公开定位包括：
  - 多发危重创伤救治
  - 心跳呼吸骤停、中毒、意外伤害救治
  - 各类疑难复杂急危重症首诊与留观
  - 突发公共事件医疗应急救援
- 官网写明其为中国创伤联盟单位、杭州市 120 救治点。

## 7. 肿瘤内科

- 公开专家页显示，牟海波担任肿瘤内科主任。
- 公开方向包括：
  - 肺癌、胃肠肿瘤、肝胆胰肿瘤的综合治疗
  - 恶性肿瘤精准免疫靶向治疗
  - 复发耐药疑难病例处理

## 8. 精神卫生科

- 官网写明该科成立于 2016 年 7 月，是树兰医院特色学科之一。
- 专家包括蔡巍、李旭娟。
- 适合用于承接睡眠、情绪、心理咨询等患者问答。

## 9. 从预约挂号页可提炼出的专病门诊能力

官网预约入口显示，树兰医院不仅有常规专科，还设置了大量专病门诊，可作为知识库的高价值检索点：

- 间质性肺疾病专科门诊
- 哮喘和慢性阻塞性肺疾病专科门诊
- 肺小结节联合门诊
- 肿瘤筛查咨询门诊
- 肿瘤热疗门诊
- 肝胰移植门诊
- 胆囊结石专科门诊
- 血栓门诊、抗凝门诊、下肢缺血专科门诊、颈动脉疾病专科门诊
- 糖尿病专科门诊、糖尿病足病内分泌门诊
- 微生态诊疗中心门诊
- 健康管理 MDT 门诊
- 戒烟门诊、国际门诊

## 10. 建议的问答检索映射

- 肝病 / 人工肝 / 发热 / 感染 / 菌群：优先匹配 `感染病科`、`肝病暨感染科门诊`
- 肝移植 / 胆胰疾病 / ERCP / EUS / 胰腺肿瘤：优先匹配 `肝胆胰外科`
- 肾移植 / 血透 / 腹透 / CRRT：优先匹配 `肾脏病科`
- 肺部感染 / 慢阻肺 / 哮喘 / 肺结节：优先匹配 `呼吸与危重症医学科`
- 休克 / 多器官衰竭 / 术后重症 / ECMO 支持类问题：优先匹配 `重症医学科`
- 肺癌 / 胃肠肿瘤 / 肝胆胰肿瘤药物治疗：优先匹配 `肿瘤内科`
- 睡眠 / 焦虑 / 抑郁 / 心理咨询：优先匹配 `精神卫生科`
