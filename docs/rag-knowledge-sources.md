## 使用注意

- 这些资料用于 RAG 引用和导诊辅助，不应让模型直接给出诊断结论或替代处方。
- 新增 PDF 会增加首次启动解析、切片和 embedding 构建时间；当前内置知识目录约 56MB，其中部分食养指南 PDF 接近 15MB。如果线上节点关闭 embedding 构建，需要先在本地完成索引构建后再同步。
- 后续若继续扩充，优先选择国家卫健委、国家疾控局、国家药监局、中华医学会公开指南或医院自身公开服务资料。
